## Reporting a Vulnerability

If you believe you have found a security vulnerability in NoirForge, please report it responsibly.

- Do **not** open a public GitHub issue for security reports.
- Instead, contact the maintainers privately with:
  - A clear description of the issue and impact
  - Reproduction steps or a proof-of-concept
  - Suggested mitigation (if known)

## Supported Versions

This repository is currently under active development and does not provide long-term support guarantees.

## Disclosure Process

We aim to:

- Acknowledge receipt of the report
- Investigate and assess severity
- Prepare and publish a fix
- Coordinate disclosure timing when appropriate
